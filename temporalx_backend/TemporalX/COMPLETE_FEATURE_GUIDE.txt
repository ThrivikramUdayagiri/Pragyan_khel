## 4. VIDEO REPAIR TOOL ğŸ”§

### What It Does
Automatically fixes temporal errors in videos using advanced algorithms.

### Repair Methods

**Frame Drop Repair**
- **Simple Duplication**: Copy previous frame
- **Frame Interpolation**: Blend previous and next frames
- **Optical Flow Interpolation**: Motion-aware frame synthesis

**Frame Merge Repair**
- **Remove Duplicates**: Skip duplicate frames
- **Adaptive Threshold**: Configure similarity threshold

**Additional Tools**
- **Variable-to-Constant Frame Rate**: Convert VFR to CFR
- **Duplicate Frame Removal**: Clean up similar frames

### How to Use
1. Analyze a video first
2. Go to "ğŸ”§ Tools" tab
3. Select "ğŸ”§ Repair Video"
4. Configure:
   - Fix Frame Drops (Yes/No)
   - Fix Frame Merges (Yes/No)
   - Interpolate Missing Frames (recommended: Yes)
5. Click "ğŸ”§ Repair Video"
6. Review repair statistics
7. Download repaired video

### Repair Statistics
- Frames Added (for drops)
- Frames Removed (for merges)
- Total Errors Fixed
- Before/After comparison

### Use Cases
- Production video cleanup
- Archive restoration
- Streaming preparation
- Re-encoding optimization

### Technical Details
- **Interpolation Methods**:
  - Weighted blending (alpha = 0.5)
  - Optical flow-based motion estimation
  - Farneback algorithm for flow computation
- **Quality Preservation**:
  - No re-encoding unless necessary
  - Maintains resolution and FPS
  - Preserves audio (if present)

### Limitations
- Cannot recover truly lost content
- Interpolation is an approximation
- Best for minor errors (1-2 frame gaps)
- Review results carefully before production use

## TIPS & BEST PRACTICES

### Performance Tips
1. **Resolution**: Lower processing resolution for faster results (640px recommended)
2. **Auto-tune**: Enable for adaptive thresholds (recommended for most videos)
3. **Parallel Processing**: Use 4 workers for batch processing on modern CPUs
4. **Progress Window**: Disable for faster processing (enable for debugging)

### Quality Tips
1. **Use Presets**: Start with appropriate preset (Static Camera, Sports, etc.)
2. **Review Confidence**: Focus on high-confidence errors (>0.8)
3. **Check Context**: Use error clips to verify detections
4. **Compare Metrics**: Cross-reference flow, SSIM, and histogram

### Workflow Tips
1. **Start Simple**: Single video analysis first
2. **Generate PDF**: Always generate PDF for records
3. **Extract Clips**: Review error clips before bulk actions
4. **Test Repair**: Try repair on copy, verify results
5. **Batch Process**: Use for recurring tasks (daily QA)

## TROUBLESHOOTING

### Common Issues

**Issue: Timeline not updating**
- Solution: Wait for UI refresh (updates every 3 frames)
- Check browser console for errors

**Issue: PDF generation fails**
- Solution: Ensure matplotlib installed correctly
- Check disk space for output

**Issue: Clip extraction returns no clips**
- Solution: No errors detected (good!), or errors below threshold

**Issue: Repair doesn't improve quality**
- Solution: Some errors can't be automatically fixed
- Review interpolation settings

**Issue: Batch processing slow**
- Solution: Reduce parallel workers or use sequential
- Lower processing resolution

**Issue: Out of memory**
- Solution: Process fewer videos at once
- Lower resolution
- Close other applications
