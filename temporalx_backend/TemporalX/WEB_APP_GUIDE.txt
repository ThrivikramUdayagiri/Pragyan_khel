NETWORK ACCESS
================================================================

By default, the web app runs on:
    http://localhost:8501

Only accessible from your computer.

To allow network access from other devices:
    streamlit run web_app.py --server.address 0.0.0.0

Then access from other devices using your IP address.

================================================================
STOPPING THE WEB APP
================================================================

To stop the server:
1. Go to the command window
2. Press Ctrl+C
3. Confirm with 'Y' if prompted

Or just close the command window.

================================================================
TROUBLESHOOTING
================================================================

Problem: Port 8501 already in use
Solution: Stop other Streamlit instances or use a different port:
          streamlit run web_app.py --server.port 8502

Problem: Browser doesn't open automatically
Solution: Manually open: http://localhost:8501

Problem: "Module not found" errors
Solution: Make sure you're using the batch file launchers

Problem: Video won't upload
Solution: Check file format (MP4, AVI, MOV, MKV, FLV, WMV)
          and file size (large files may take time to upload)

Problem: Processing is slow
Solution: Reduce "Processing Resolution" in sidebar
          or use a shorter video clip for testing
