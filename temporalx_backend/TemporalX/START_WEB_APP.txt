================================================================
                   TROUBLESHOOTING
================================================================

Problem: Browser doesn't open
Solution: Manually go to http://localhost:8501

Problem: "Address already in use"
Solution: Another instance is running. Close it first.

Problem: Video upload fails
Solution: Check file format and size. Large files take time.

Problem: Processing is slow
Solution: In sidebar, reduce "Processing Resolution" to 480

                  ACCESSING FROM OTHER DEVICES

   .venv\Scripts\streamlit.exe run web_app.py --server.address 0.0.0.0

   ipconfig (Windows)

3. On other devices, open browser to:
   http://YOUR_IP_ADDRESS:8501
